# Продуктовая аналитика: Анализ вовлеченности и роста аудитории

## О проекте
Цель проекта — оценить «здоровье» образовательной платформы через метрики активности (DAU/MAU) и проанализировать эффективность удержания пользователей.

## Стек технологий
* **SQL (DBeaver):** Оконные функции, сложные JOIN, агрегация метрик активности.
* **Power BI:** Построение интерактивного дашборда, моделирование связей между регистрациями и визитами.

## Ключевые показатели (Insights)
* **Рост базы:** За 4 месяца база пользователей выросла с 0.5k до 2.4k человек.
* **Вовлеченность:** Выявлен «эффект ножниц» — накопленная база растет быстрее, чем ежемесячная активность (MAU), что сигнализирует о необходимости реактивации «спящих» юзеров.
* **Sticky Factor (Липкость):** Стабильный показатель **27.5%** указывает на наличие лояльного ядра аудитории, использующего сервис почти ежедневно.
